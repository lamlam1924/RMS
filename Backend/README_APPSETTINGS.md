# Cấu hình appsettings

Để chạy project, tạo file `appsettings.json` từ template:

```bash
cp appsettings.Example.json appsettings.json
```

Sau đó sửa `appsettings.json` và thay các giá trị placeholder bằng thông tin thật (Google OAuth, Database, Email...).

**Lưu ý:** Không commit `appsettings.json` lên Git vì chứa thông tin nhạy cảm.
